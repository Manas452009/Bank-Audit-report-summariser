Test Upload to http://localhost:8000/process-pdf

This simulates the Summarizer component's upload behavior

What was implemented:

  • src/components/Summarizer.tsx - Added upload state (isUploading, uploadResult) and uploadFile() function
  • src/components/Summarizer.css - Added styles for uploading status and result display
  • vite.config.ts - Added proxy for /process-pdf to http://localhost:8000

How the upload works:

  1. User selects/drops a file → file validation runs
  2. If valid → uploadFile() creates FormData with the file
  3. POST to http://localhost:8000/process-pdf with FormData
  4. Response JSON is displayed in the upload-result section
